> Big-O notation(빅오 표기법)

### 개념
```
Time Complexity(시간 복잡도)
알고리즘의 수행시간 분석 결과 >> 속도

Space Complexity(공간 복잡도)
메모리 사용량 분석 결과

보통, 시간 복잡도가 공간 복잡도에 비해 훨씬 중요. 다시 말해, 실행속도가 중요

EX)
O(1) < O(logN) < O(N) < O(N * logN) < O(N^2) < O(N^3) < O(2^N) < O(N!)
```

### COMMENT
* 윤성우의 열혈 자료구조
* 프로그래밍 대회에서 배우는 알고리즘 문제해결전략
