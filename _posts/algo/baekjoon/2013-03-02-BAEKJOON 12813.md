> 이진수 연산 [GO 12813]

### CODE
```java
class Main {
	public static void main(String[] args) throws Exception {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		String[] a = br.readLine().split("");
		String[] b = br.readLine().split("");
		String[] c = new String[a.length];

		for (int i = 0; i < a.length; i++)
			c[i] = a[i].equals("1") && b[i].equals("1") ? "1" : "0";
		for (String str : c)
			System.out.print(str);
		System.out.println();

		for (int i = 0; i < a.length; i++)
			c[i] = a[i].equals("1") || b[i].equals("1") ? "1" : "0";
		for (String str : c)
			System.out.print(str);
		System.out.println();

		for (int i = 0; i < a.length; i++)
			c[i] = !a[i].equals(b[i]) ? "1" : "0";
		for (String str : c)
			System.out.print(str);
		System.out.println();

		for (int i = 0; i < a.length; i++)
			c[i] = a[i].equals("1") ? "0" : "1";
		for (String str : c)
			System.out.print(str);
		System.out.println();

		for (int i = 0; i < a.length; i++)
			c[i] = b[i].equals("1") ? "0" : "1";
		for (String str : c)
			System.out.print(str);
		System.out.println();
	}
}
```
