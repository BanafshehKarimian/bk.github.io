> 드래곤 앤 던전 [GO 16434]

### CODE
```java
class Main {
	static int n;
	public static void main(String[] args) throws Exception {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st;
		st = new StringTokenizer(br.readLine());
		n = Integer.parseInt(st.nextToken());
		int startattack = Integer.parseInt(st.nextToken());
		int[][] input = new int[3][n];
		for (int j = 0; j < n; j++) {
			st = new StringTokenizer(br.readLine());
			for (int i = 0; i < 3; i++)
				input[i][j] = Integer.parseInt(st.nextToken());
		}
		System.out.print(binarySearch(input, 0, ((long)2 << 62) - 1, startattack));
	}

	public static long binarySearch(int[][] input, long bottom, long top, int startattack) {
		while (bottom <= top) {
			long mid = (bottom + top) / 2;
			if (check(input, mid, startattack))
				bottom = mid + 1;
			else
				top = mid - 1;
		}
		return bottom;
	}

	public static boolean check(int[][] input, long h, int startattack) {
		long attack = startattack;
		long hp = h;
		boolean tf = false;
		for (int j = 0; j < n; j++) {
			if (input[0][j] == 1) {
				if (input[2][j] % attack == 0)
					hp -= (input[2][j] / attack - 1) * input[1][j];
				else
					hp -= input[2][j] / attack * input[1][j];
				if (hp <= 0){
					tf = true;
					break;
				}				
			}
			else {
				hp += input[2][j];
				hp = hp > h ? h : hp;
				attack += input[1][j];
			}
		}
		return tf;
	}
}
```

### COMMENT
* Divide and Conquer
* Binary Search
* long 변수 활용(int 형시 같은 코드라도 오답 가능성)
