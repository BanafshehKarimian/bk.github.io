> DFS와 BFS(1) 인접 행렬 : [GO 1260]

### CODE

```java
public class Main {
	public static int[][] numbers;
	public static boolean[] check;
	public static int n, m, start;
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		n = sc.nextInt();
		m = sc.nextInt();
		start = sc.nextInt();
		numbers = new int [n+1][n+1];

		for (int i = 0; i < m; i++) {
			int u = sc.nextInt();
			int v = sc.nextInt();
			numbers[u][v] = 1;
			numbers[v][u] = 1;
		}

		check = new boolean[n + 1];
		dfs(start);

		System.out.println();
		check = new boolean[n + 1];
		bfs(start);
	}

	public static void dfs(int x) {
		if (check[x])
			return;
		check[x] = true;
		System.out.print(x + " ");
		for (int i = 1; i < n+1; i++) {
			if (numbers[x][i] == 1 && !check[i])
				dfs(i);
		}
	}

	public static void bfs(int x) {
		Queue<Integer> q = new LinkedList<Integer>();
		q.add(x);
		check[x] = true;
		while (!q.isEmpty()) {
			int temp = q.remove();
			System.out.print(temp + " ");
			for (int i = 0; i < n+1; i++) {
				if (numbers[temp][i] == 1 && !check[i]) {
					q.add(i);
					check[i] = true;
				}
			}
		}
	}
}
```
### COMMENT
* 인접 행렬
* O(V^2)
