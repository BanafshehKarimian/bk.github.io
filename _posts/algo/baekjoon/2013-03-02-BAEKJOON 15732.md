> 도토리 숨기기 [GO 15732]

### CODE
```java
class Main {
	public static void main(String[] args) throws Exception {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st;
		st = new StringTokenizer(br.readLine());
		int n = Integer.parseInt(st.nextToken());
		int k = Integer.parseInt(st.nextToken());
		int d = Integer.parseInt(st.nextToken());
		Rule[] input = new Rule[k];
		for (int i = 0; i < k; i++) {
			st = new StringTokenizer(br.readLine());
			input[i] = new Rule(Integer.parseInt(st.nextToken()), Integer.parseInt(st.nextToken()),
					Integer.parseInt(st.nextToken()));
		}
		System.out.print(binarySearch(input, 0, n, d));
	}

	public static long binarySearch(Rule[] input, int bottom, int top, int d) {
		while (bottom <= top) {
			int mid = (bottom + top) / 2;
			if (check(input, mid, d))
				bottom = mid + 1;
			else
				top = mid - 1;
		}
		return bottom;
	}

	public static boolean check(Rule[] input, int h, int d) {
		long count = 0;
		for (int i = 0; i < input.length; i++) {
			if (input[i].start > h)
				continue;
			else if (input[i].end <= h)
				count += (input[i].end - input[i].start) / input[i].size + 1;
			else
				count += (h - input[i].start) == 0 ? 1 : (h - input[i].start) / input[i].size + 1;
		}
		if (count < d)
			return true;
		return false;
	}

	static class Rule {
		int start;
		int end;
		int size;
		Rule(int start, int end, int size) {
			this.start = start;
			this.end = end;
			this.size = size;
		}
	}
}
```

### COMMENT
* Divide and Conquer
* Binary Search
* long 변수 활용(int 형시 같은 코드라도 오답 가능성)
