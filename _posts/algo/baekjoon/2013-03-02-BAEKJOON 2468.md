> 영역 구하기 : [GO 2468]

### CODE

```java
public class Main {
	public static int[] dx = { 0, 0, 1, -1 };
	public static int[] dy = { 1, -1, 0, 0 };
	public static int[][] numbers, temp, result;
	public static int count = 0;

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int n = sc.nextInt();
		numbers = new int[n][n];
		temp = new int[n][n];
		result = new int[n][n];
		int max = 1;
		for (int j = 0; j < n; j++)
			for (int i = 0; i < n; i++)
				numbers[i][j] = sc.nextInt();

		for (int i = 1; i <= 100; i++) {
			setting(numbers, temp, i, n, n);
			int tmp = solve(temp, result, 0, n, n);
			if (tmp > max)
				max = tmp;
		}
		System.out.println(max);
	}

	public static void setting(int[][] numbers, int[][] temp, int number, int m, int n) {
		for (int j = 0; j < m; j++) {
			for (int i = 0; i < n; i++) {
				if (number >= numbers[i][j])
					temp[i][j] = 0;
				else
					temp[i][j] = numbers[i][j];
                result[i][j] = 0;
			}
		}
	}

	public static int solve(int[][] temp, int[][] result, int count, int m, int n) {
		for (int j = 0; j < m; j++)
			for (int i = 0; i < n; i++)
				if (temp[i][j] != 0 && result[i][j] == 0)
					dfs(temp, result, i, j, ++count, m, n);
		return count;
	}

	public static void dfs(int[][] temp, int[][] result, int x, int y, int count, int m, int n) {
		result[x][y] = count;
		for (int i = 0; i < 4; i++) {
			int nx = x + dx[i];
			int ny = y + dy[i];
			if (nx >= 0 && nx < n && ny >= 0 && ny < m)
				if (temp[nx][ny] != 0 && result[nx][ny] == 0)
					dfs(temp, result, nx, ny, count, m, n);
		}
	}
}
```
### COMMENT
* 인접 행렬 + DFS
* dx, dy : 상하좌우 이동을 위한 배열
* temp : 물에 잠기는 높이 반영한 배열
* result : 안전영역 결과물 배열
* 주의 >> 최소값 1(모두 안잠기는 경우)
