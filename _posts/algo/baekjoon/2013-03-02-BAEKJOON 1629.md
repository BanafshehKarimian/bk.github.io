> BAE/<JOON 곱셈 [GO 1629]

### CODE
```java
public class Main {
	public static void print(int a, int b, int c) {
		System.out.println(pow(a, b, c));
	}

	public static long pow(int a, int b, int c) {
		if (b == 0)
			return 1;

		long n = pow(a, b / 2, c);
		long temp = n * n % c;

		if (b % 2 == 0)
			return temp;
		else
			return a * temp % c;
	}

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int a = sc.nextInt();
		int b = sc.nextInt();
		int c = sc.nextInt();
		print(a, b, c);
	}
}

```
