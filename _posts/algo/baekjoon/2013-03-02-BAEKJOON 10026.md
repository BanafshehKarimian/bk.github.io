> 적록색약 : [GO 10026]

### CODE

```java
public class Main {
	public static int[] dx = { 0, 0, 1, -1 };
	public static int[] dy = { 1, -1, 0, 0 };
	public static String[][] numbers;
	public static int[][] temp;

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int n = sc.nextInt();
		sc.nextLine();

		numbers = new String[n + 1][n + 1];
		temp = new int[n + 1][n + 1];
		for (int i = 0; i < n; i++) {
			String s = sc.nextLine();
			for (int j = 0; j < n; j++)
				numbers[i][j] = s.substring(j, j + 1);
		}
		System.out.print(solve(numbers, temp, 0, n, n) + " ");

		for (int i = 0; i < n; i++)
			for (int j = 0; j < n; j++)
				if (numbers[i][j].equals("R"))
					numbers[i][j] = "G";
		for (int i = 0; i < n; i++)
			for (int j = 0; j < n; j++)
					temp[i][j] = 0;
		System.out.print(solve(numbers, temp, 0, n, n));
	}

	public static int solve(String[][] numbers, int[][] temp, int count, int m, int n) {
		for (int j = 0; j < n; j++)
			for (int i = 0; i < m; i++){
				String str = numbers[i][j];
				if (numbers[i][j].equals(str) && temp[i][j] == 0)
					dfs(numbers, temp, i, j, str, ++count, m, n);
			}
		return count;
	}

	public static void dfs(String[][] numbers, int[][] temp, int x, int y, String str, int count, int m, int n) {
		temp[x][y] = count;
		for (int i = 0; i < 4; i++) {
			int nx = x + dx[i];
			int ny = y + dy[i];
			if (nx >= 0 && nx < m && ny >= 0 && ny < n )
				if (numbers[nx][ny].equals(str) && temp[nx][ny] == 0)
					dfs(numbers, temp, nx, ny, str, count, m, n);
		}
	}
}
```
### COMMENT
* 인접 행렬 + DFS
* dx, dy : 상하좌우 이동을 위한 배열
* temp : 적록색약 결과물 배열
