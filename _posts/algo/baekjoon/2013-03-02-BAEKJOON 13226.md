> Divisors Again : [GO 13226]

### CODE

```java
public class Main {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int c = sc.nextInt();
		for (int i = 0; i < c; i++) {
			int l = sc.nextInt();
			int u = sc.nextInt();
			int max = 0;
			for (int k = l; k <= u; k++) {
				int count = 0;
				for (int j = 1; j * j <= k; j++)
					if (k % j == 0) {
						count++;
						if (j * j < k)
							count++;
					}
				if (count > max)
					max = count;
			}
			System.out.println(max);
		}
	}
}
```
