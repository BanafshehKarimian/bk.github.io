> 암호 만들기 : [GO 1759]

### CODE

```java
public class Main {
	public static int l, c;
	public static String[] input;
	public static String[] answer;
	public static ArrayList<String> condition = new ArrayList<>();

	public static void main(String[] args) {
		condition.add("a");
		condition.add("e");
		condition.add("u");
		condition.add("i");
		condition.add("o");

		Scanner sc = new Scanner(System.in);
		l = sc.nextInt();
		c = sc.nextInt();
		sc.nextLine();
		answer = new String[l];
		input = sc.nextLine().split(" ");
		Arrays.sort(input);
		dfs(0, 0);
	}

	public static void dfs(int start, int size) {
		if (size == l) {
			if (check())
				print();
			return;
		}
		for (int i = start; i < c; i++) {
			answer[size] = input[i];
			dfs(i+1, size+1);
		}
	}

	public static boolean check() {
		int count1 = 0;
		int count2 = 0;
		for (String s : answer) {
			if (condition.contains(s))
				count1 += 1;
			else
				count2 += 1;
		}
		return count1 >= 1 && count2 >= 2;
	}

	public static void print() {
		for (int i = 0; i < answer.length; i++)
			System.out.print(answer[i]);
		System.out.println();
	}
}
```
### COMMENT
* 문자열 조합 + DFS
* check() : 자음, 모음 조건 확인
* contains() : ArrayList 안에 값이 있는 확인
