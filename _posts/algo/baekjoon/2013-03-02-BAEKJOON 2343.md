> 기타 레슨 [GO 2343]

### CODE
```java
class Main {
	public static void main(String[] args) throws Exception {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st;
		st = new StringTokenizer(br.readLine());
		int n = Integer.parseInt(st.nextToken());
		int m = Integer.parseInt(st.nextToken());
		int[] input = new int[n];
		st = new StringTokenizer(br.readLine());
		for (int i = 0; i < n; i++)
			input[i] = Integer.parseInt(st.nextToken());
		System.out.print(binarySearch(input, 0, 1000000000, m));
	}

	public static long binarySearch(int[] input, int bottom, int top, int m) {
		while (bottom <= top) {
			int mid = (bottom + top) / 2;
			if (check(input, mid, m))
				bottom = mid + 1;
			else
				top = mid - 1;
		}
		return bottom;
	}

	public static boolean check(int[] input, int mid, int m) {
		long count = 1;
		long sum = 0;
		boolean tf = false;
		for (int i = 0; i < input.length; i++) {
			if (input[i] > mid) {
				tf = true;
				break;
			}
			sum += input[i];
			if (sum > mid) {
				sum = input[i];
				count++;
			}
			if (count > m) {
				tf = true;
				break;
			}
		}
		return tf;
	}
}
```

### COMMENT
* Divide and Conquer
* Binary Search
* long 변수 활용(int 형시 같은 코드라도 오답 가능성)
* 예외 처리
