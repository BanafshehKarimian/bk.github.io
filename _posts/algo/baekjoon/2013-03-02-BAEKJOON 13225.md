> Divisors : [GO 13225]

### CODE

```java
public class Main {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int c = sc.nextInt();
		for (int i = 0; i < c; i++) {
			int n = sc.nextInt();
			int count = 0;
			for (int j = 1; j * j <= n; j++)
				if (n % j == 0) {
					count++;
					if (j * j < n)
						count++;
				}
			System.out.println(n + " " + count);
		}
	}
}
```
