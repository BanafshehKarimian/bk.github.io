> 한수 : [GO 1065]

### CODE

```java
public static int solve(int n) {
	int answer = 0;
	for (int i = 1; i < n + 1; i++) {
		int temp = i;
		ArrayList<Integer> res = new ArrayList<>();
		if (i == 1000)
			break;
		if (i < 100) {
			answer++;
			continue;
		}
		while (temp != 0) {
			res.add(temp % 10);
			temp = temp / 10;
		}
		if (res.get(0) - res.get(1) == res.get(1) - res.get(2))
			answer++;
	}
	return answer;
}
```
### COMMENT
* 1 <= 한수범위 <= 1000
