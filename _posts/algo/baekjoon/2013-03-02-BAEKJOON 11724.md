> 연결 요소의 개수 : [GO 11724]

### CODE

```java
public class Main {
	public static ArrayList<Integer>[] numbers;
	public static boolean[] check;
	public static int answer = 0;

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int n = sc.nextInt();
		int m = sc.nextInt();

		numbers = (ArrayList<Integer>[]) new ArrayList[n + 1];
		for (int i = 1; i <= n; i++)
			numbers[i] = new ArrayList<Integer>();

		for (int i = 0; i < m; i++) {
			int u = sc.nextInt();
			int v = sc.nextInt();
			numbers[u].add(v);
			numbers[v].add(u);
		}

		for (int i = 1; i <= n; i++)
			Collections.sort(numbers[i]);

		check = new boolean[n + 1];
		solve(n);
	}

	public static void solve(int n) {
		for (int i = 1; i < n+1; i++) {
			if (!check[i]) {
				dfs(i);				
				answer++;
			}
		}
		System.out.println(answer);
	}

	public static void dfs(int start) {
		if (check[start])
			return;
		check[start] = true;
		for (int y : numbers[start])
			if (!check[y])
				dfs(y);
	}
}
```
### COMMENT
* 인접 리스트 + DFS
