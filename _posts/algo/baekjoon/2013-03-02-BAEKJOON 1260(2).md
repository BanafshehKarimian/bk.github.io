> DFS와 BFS(2) 인접 리스트 : [GO 1260]

### CODE

```java
public class Main {
	public static ArrayList<Integer>[] numbers;
	public static boolean[] check;
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int n = sc.nextInt();
		int m = sc.nextInt();
		int start = sc.nextInt();

		numbers = (ArrayList<Integer>[]) new ArrayList[n + 1];
		for (int i = 1; i <= n; i++)
			numbers[i] = new ArrayList<Integer>();

		for (int i = 0; i < m; i++) {
			int u = sc.nextInt();
			int v = sc.nextInt();
			numbers[u].add(v);
			numbers[v].add(u);
		}

		for (int i = 1; i <= n; i++)
			Collections.sort(numbers[i]);

		check = new boolean[n + 1];
		dfs(start);

		System.out.println();
		check = new boolean[n + 1];
		bfs(start);
	}

	public static void dfs(int start) {
		if (check[start])
			return;
		check[start] = true;
		System.out.print(start + " ");
		for (int y : numbers[start]) {
			if (!check[y])
				dfs(y);
		}
	}

	public static void bfs(int start) {
		Queue<Integer> q = new LinkedList<Integer>();
		q.add(start);
		check[start] = true;
		while (!q.isEmpty()) {
			int temp = q.remove();
			System.out.print(temp + " ");
			for (int y : numbers[temp]) {
				if (!check[y]) {
					q.add(y);
					check[y] = true;
				}
			}
		}
	}
}
```
### COMMENT
* 인접 리스트
* O(V+E)
