> 섬의 개수 : [GO 4963]

### CODE

```java
public class Main {
	public static int h, w;
	public static int[][] numbers;
	public static int[][] temp;
	public static int[] dx = { 0, 0, 1, -1, 1, 1, -1, -1 };
	public static int[] dy = { 1, -1, 0, 0, 1, -1, 1, -1 };

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		while (true) {
			w = sc.nextInt();
			h = sc.nextInt();
			if (w == 0 && h == 0)
				break;
			numbers = new int[w][h];
			temp = new int[w][h];
			for (int j = 0; j < h; j++)
				for (int i = 0; i < w; i++)
					numbers[i][j] = sc.nextInt();

			int count = 0;
			for (int j = 0; j < h; j++)
				for (int i = 0; i < w; i++)
					if (numbers[i][j] == 1 && temp[i][j] == 0)
						dfs(numbers, temp, i, j, w, h, ++count);
			System.out.println(count);
		}
	}

	public static void dfs(int[][] numbers, int[][] temp, int x, int y, int w, int h, int count) {
		temp[x][y] = count;
		for (int i = 0; i < 8; i++) {
			int nx = x + dx[i];
			int ny = y + dy[i];
			if (nx >= 0 && ny >= 0 && nx < w && ny < h)
				if (numbers[nx][ny] == 1 && temp[nx][ny] == 0)
					dfs(numbers, temp, nx, ny, w, h, count);
		}
	}
}
```
### COMMENT
* 인접 행렬 + DFS
* dx, dy : 8방향(상하좌우 + 대각선 4방향) 이동을 위한 배열
* temp : 섬 결과물 배열
