> 단지번호붙이기 : [GO 2667]

### CODE

```java
public class Main {
	public static int[] dx = { 0, 0, 1, -1 };
	public static int[] dy = { 1, -1, 0, 0 };
	public static int[][] numbers, temp;
	public static int[] result;
	public static int n;
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		n = sc.nextInt();
		sc.nextLine();

		numbers = new int[n][n];
		temp = new int[n][n];

		for (int i = 0; i < n; i++) {
			String s = sc.nextLine();
			for (int j = 0; j < n; j++)
				numbers[i][j] = Integer.parseInt(s.substring(j, j + 1));
		}

		result = new int[solve(numbers, temp, 0, n)];
		print(result, temp, n);
	}

	public static int solve(int[][] numbers, int[][] temp, int count, int n) {
		for (int i = 0; i < n; i++)
			for (int j = 0; j < n; j++)
				if (numbers[i][j] == 1 && temp[i][j] == 0)
					dfs(numbers, temp, i, j, ++count, n);
		return count;
	}

	public static void dfs(int[][] numbers, int[][] temp, int x, int y, int count, int n) {
		temp[x][y] = count;
		for (int i = 0; i < 4; i++) {
			int nx = x + dx[i];
			int ny = y + dy[i];
			if (nx >= 0 && nx < n && ny >= 0 && ny < n) {
				if (numbers[nx][ny] == 1 && temp[nx][ny] == 0) {
					dfs(numbers, temp, nx, ny, count, n);
				}
			}
		}
	}

	public static void print(int[] result, int[][] temp, int n) {
		for (int i = 0; i < n; i++)
			for (int j = 0; j < n; j++)
				if (temp[i][j] != 0)
					result[temp[i][j] - 1] += 1;

		Arrays.sort(result);
		System.out.println(result.length);
		for (int i = 0; i < result.length; i++)
			System.out.println(result[i]);
	}
}
```
### COMMENT
* 인접 행렬 + DFS
* dx, dy : 상하좌우 이동을 위한 배열
* temp : 단지번호 결과물 배열
* result : 결과물 출력 배열
* 유사문제 [GO 1012]
