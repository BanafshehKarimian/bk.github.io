> 카펫 : [GO 42842]

### CODE
```java
class Solution {
	public int[] solution(int brown, int red) {
		int[] answer = new int[2];

		for (int i = 1; i <= (int) Math.sqrt(red); i++) {
			if (red % i == 0) {
				if (brown == (2 * i) + (2 * red / i) + 4) {
					answer[0] = red / i + 2;
					answer[1] = i + 2;
					break;
				}
			}
		}

		return answer;
	}
}
```
